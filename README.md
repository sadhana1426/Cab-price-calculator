# Cab Fare Calculator (Python)

A simple Python GUI application built using **Tkinter** that calculates the fare for different cab types based on travel distance.
The application also supports **speech recognition** for booking a cab.

## Features

* Select different cab types (Standard, Luxury, SUV)
* Enter travel distance
* Automatic fare calculation
* Speech recognition booking
* Simple GUI using Tkinter

## Technologies Used

* Python
* Tkinter
* SpeechRecognition library

## How to Run

1. Install required library

```
pip install SpeechRecognition
```

2. Run the program

```
python calculator.py
```

## Example

User selects cab type and distance → Program calculates total fare.
